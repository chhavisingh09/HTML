FAQ (Frequently Asked Questions)

1. What is HTML?

HTML stands for Hypertext Markup Lanaguage. It is used for creating webpages.

2. What is a tag in HTML?

Tag is a keyword used to display somethinng in HTML.

3. What is an Attribute?

An Attribute gives extra information to a tag.

4. Do we need CSS to use HTML?

No, HTML works alone. CSS is only for Styling.
